# CrowdSense AI

CrowdSense AI is a real-time crowd behavior analysis and anomaly detection system using OpenCV and Machine Learning.

## Features

- Real-time person detection
- Multi-person tracking
- Crowd density analysis
- Zone-based monitoring
- Anomaly detection using machine learning
- Alert logging
- Flask dashboard

## Tech Stack

- Python
- OpenCV
- NumPy
- Scikit-learn
- Flask
- PyYAML
